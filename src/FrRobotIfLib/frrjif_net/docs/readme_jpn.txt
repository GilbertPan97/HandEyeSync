FANUC ROBOT Interface


  FANUC ROBOT Interfaceは、イーサネット上でロボットのデータにア
セスするためのランタイムモジュールです。


- 動作環境
  robotif_jpn.pdfをご覧ください。


- 仕様書
  robotif_dot_net_jpn.pdfをご覧ください。


- ランタイムモジュール
　RobotInterfaceDotNet.dllおよびbridgeRobotIF.dllをご使用ください。


- サンプルプログラム
  実際にFANUC ROBOT Interfaceのプログラミング方法をご理解いただくには、サンプルプログラムを見て
いただくのがよいと考えます。
  サンプルプログラムについては、SampleCode\readme_jpn.txt をご覧ください。
